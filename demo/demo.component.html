Cloudinary image directive

Initialize your app by creating a module called DemoModule

        
            {{ CODE_SAMPLES.module }}

Then create the Demo component with CloudinaryOptions. The cloud_name property is required.

        
            {{ CODE_SAMPLES.components['cl-image']}}

For example, displaying the image with a public ID of sample uploaded to Cloudinary's demo account in jpg format:

        
            {{ CODE_SAMPLES.templates['cl-image'].simple}}

Displaying the same image with a limited height of 200 pixels

        
            {{ CODE_SAMPLES.templates['cl-image']['transform-height']}}

Image transformations are limited to the following parameters in the current version:
  • height, width: in pixels only (aspect ratio might be supported in a futur version)
  • crop
  • gravity, x, y

For example, displaying the same image cropped to a width of 300 pixels and a height of 200 pixels.

        
            {{ CODE_SAMPLES.templates['cl-image']['transform-crop']}}

Another example, displaying an image with automatic cropping and fill mode.

        
            {{ CODE_SAMPLES.templates['cl-image']['transform-autocrop']}}
